Execution Instructions:
	to run the program you must have text files in the same directory as the src bin and include folders
(along with data1 and data2). First type make. Second type ./bin/run textfile 1(which will be for the inversions)
textfile 2(is for the convex hull). For a trial of the program you can type the following, make > 
./bin/run data1.txt data2.txt.
